## Description

[Nest](https://github.com/nestjs/nest) Backend for a test application.

## Installation

```bash
$ npm install
```

## Running the app

You need Docker installed and running.

```bash
# Run postgres server
$ docker-compose up

```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
